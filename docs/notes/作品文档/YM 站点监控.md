## ::mingcute:document-line:: 简介

==YM 站点监控== 是一个由 [YOU MING 柚明](/notes/更多/工作室.html#you-ming-柚明) 于 ==2025.4.10=={.info} 开始制作的一个 ==网页项目=={.info} 。

- 一个基于 UptimeRobot API 的在线状态面板 | 站点监测 | 状态检测
- 网站用于给观众实时显示网站可用性，并记录历史数据
