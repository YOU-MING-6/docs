::: info YM 审核
- 审核时间：节假日及周末
- 如果你发现你的反馈于以上时间并没有得到审核，可能是因为

  - 我们的节假日及周末与你有所差异
  - 你的邮箱填写错误
  - 我们并没有上线
  - 正在开发&制作某个项目，没空
  - 没有收到你的反馈（可能较小）

- 遇到过长时间没有审核，可尝试重新发送反馈，并仔细检查邮箱是否可用。
:::
